Proyectos Destacados

Diseño e Implementación de Red LAN/WLAN para PYME

Se realizó el diseño completo de la infraestructura de red para una oficina, incluyendo el direccionamiento IP (IPv4), configuración de VLANs para segmentar el tráfico, y la implementación de switches y puntos de acceso inalámbricos (APs) para garantizar una cobertura total y segura.

Configuración de Firewall y VPN Site-to-Site

Implementación de políticas de seguridad en un firewall de próxima generación (NGFW) para proteger la red corporativa. Se configuró una VPN IPsec para conectar de forma segura dos sucursales, permitiendo el intercambio de datos de manera encriptada a través de internet.

Monitoreo de Red con Zabbix

Despliegue de un sistema de monitoreo de red de código abierto (Zabbix) para supervisar el rendimiento y la disponibilidad de servidores y dispositivos de red críticos. Se configuraron alertas automáticas por correo electrónico para notificar fallos en tiempo real.
